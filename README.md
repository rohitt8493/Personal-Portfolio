## Project Overview

A simple, responsive front-end portfolio built with **HTML**, **CSS**, and **JavaScript**. The site demonstrates semantic HTML structure, clean CSS styling, and basic JavaScript interactivity. It includes sections for **About**, **Skills**, **Projects**, and a **Contact** form that shows a front-end confirmation message (no backend).

## Features

- Vertical single-page layout with semantic sections  
- Skills: CPP; Python; Java; JavaScript; HTML; CSS; Web Development; AI; Problem Solving  
- Projects: Real-time Sign Language for Deaf People; Academic AI Chatbot for Students (each with a short summary)  
- Contact form requiring Name, Email, Phone, Message with front-end validation and confirmation  
- JavaScript interactions: greeting alert, scroll-to-contact, form validation and confirmation
